Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit

Este projeto utilizou conceitos avançados de programação em Python e manipulação de dados, aliados ao pensamento estratégico e lógica de negócio, bem como ferramentas de desenvolvimento web, como o Streamlit e o Github, para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na Índia. O objetivo principal foi fornecer uma interface web de fácil acesso para que os gestores da empresa possam visualizar de forma clara e concisa as principais métricas e indicadores do negócio.

O resultado final do projeto foi um painel gerencial que pode ser acessado a partir de qualquer dispositivo conectado à internet, hospedado em um ambiente Cloud seguro e confiável. As principais ferramentas utilizadas incluíram o ambiente de desenvolvimento Jupyter Lab, o terminal e as bibliotecas do Python, além do Streamlit para a criação da interface web e do Github para controle de versão e colaboração.

O projeto demonstra a capacidade de aplicar conceitos de Machine Learning e Data Science em um ambiente de negócios, fornecendo uma solução de painel gerencial acessível e fácil de usar para ajudar os gestores a tomar decisões informadas e estratégicas.

As ferramentas utilizadas foram:

  • Python.
  • Jupyter Lab.
  • Github.
  • Terminal.
  • Streamlit.
  • Streamlit Cloud.
  • Visual Studio Code.
